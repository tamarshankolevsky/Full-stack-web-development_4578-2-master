# js types
##  Primitive and non-pritive types
Primitive types (built-in types in js):
* number
* boolean
* undefined
* null
* symbol
* string

Non - Primitive types:
* all the types that are not in theb list above


##  Value type and Ref type
Value types:
* number
* boolean
* undefined
* null
* symbol


Ref types:
* string
* function
* object (array, json, Date, class, etc...)
